OpenRocket simulation files, pre-flight predictions, and post-flight comparison data for every mission we've run. We publish these openly because the rocketry community gets smarter when we share not just what worked — but what we *thought* would work before launch, and where we were wrong.

We're a student rocketry association from Anand, Gujarat, building towards 100 km — one mission at a time.

## Getting Started

1. Download [OpenRocket](https://openrocket.info/) — free and cross-platform
2. Clone or download this repo
3. Open any `.ork` file directly in OpenRocket
4. Read the `analysis.md` alongside it for context

> OpenRocket version can affect file compatibility. The version used is noted in each mission folder's README.

## Contributing

If you spot a modelling assumption you'd challenge, or you've used our files and got different results — open an issue or start a Discussion. Peer review on simulation methodology is exactly why we publish these openly.

We're students. We make assumptions that don't hold. Tell us.
